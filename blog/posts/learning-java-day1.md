## Java语言
Java是一种高级语言，能在不同平台（Mac，Windows，Linux）下编译并运行

### 1.特点 

-Java是可移植的，
-Java大小写敏感

### 2. Java程序流程

创建.java文件->编写->通过javac XXX.java编译->生成 .class文件->通过 java XXX 运行


## 代码示例

HelloWorld.java
```java
public class HelloWorld{
    public static void main(String[] args){
        System.out.println("Hello World");
    }
}
```
-文件名必须和类名一致
